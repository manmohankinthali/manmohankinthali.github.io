# manmohankinthali.github.io

Personal portfolio website for **Manmohan Kinthali** – Embedded Software Engineer specializing in Automotive, AUTOSAR, Power Electronics & Functional Safety.

## Live Site

[https://manmohankinthali.github.io](https://manmohankinthali.github.io)

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Responsive single-page design
- Google Fonts (Inter)
- Hosted on GitHub Pages

## Sections

- Hero with highlights & stats
- About Me & Key Expertise
- Featured Projects
- Technical Stack
- Work Experience (Timeline)
- Education
- Contact

## Local Development

Simply open `index.html` in a browser, or use any local server:

```bash
npx serve .
```

## Deployment

Push to `main` branch — GitHub Pages will serve `index.html` automatically.
